Katana Native Interface (KNI) - Brief Install Guide (Linux/*nix only)
====================================================================

1) Unpack the archive
2) Change into the folder 'KatanaNativeInterface'
3) Type 'make' -> This will build the library and all the demo-applications 

Please read the "KNI Manual.pdf" (located in doc/) for more information.
